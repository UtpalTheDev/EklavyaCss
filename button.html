Button
An essential element of any design. It's meant to look and behave as an interactive element of your page.
Primary Button
Primary button comes with default size with black background. To use primary button you need to add primary-button class in your html.
Primary button comes with 3 different sizes are small, medium and large. To use small button you need to add primary-button sm classes in your html. To use medium button you need to add primary-button md classes in your html. To use large button you need to add primary-button lg classes in your html.

Secondary Button
Secondary button comes with default size with white background and icon. To use secondary button you need to add .secondary-button classes in your html.
Secondsry button comes with 3 different sizes are small, medium and large. To use small button you need to add secondary-button sm classes in your html. To use medium button you need to add secondary-button md classes in your html. To use large button you need to add secondary-button lg classes in your html.

Icon Button
Simple icon button comes with default size with transparent background. To use icon button you need to add icon-button classes in your html.
Icon button comes with 3 different sizes are small, medium and large. To use small button you need to add icon-button sm classes in your html. To use medium button you need to add icon-button md classes in your html. To use large button you need to add icon-button lg classes in your html.
Squared Icon Button
Squared icon button comes with default size with lightgrey background. To use squared icon button you need to add icon-button square classes in your html.
Squared icon button comes with 3 different sizes are small, medium and large. To use small button you need to add icon-button square sm classes in your html. To use medium button you need to add icon-button square md classes in your html. To use large button you need to add icon-button square lg classes in your html.
Rounded Icon Button
Rounded icon button comes with default size with lightgrey background. To use rounded icon button you need to add icon-button round classes in your html.
Rounded icon button comes with 3 different sizes are small, medium and large. To use small button you need to add icon-button round sm classes in your html. To use medium button you need to add icon-button round md classes in your html. To use large button you need to add icon-button round lg classes in your html.
Icon Button with Badge
Rounded icon button badge comes with red color background. To use batch on button you need to add icon-badge class in your div element which is dedicated to use for batch.
Floating Action Button
Floating button comes with default size with black background and stick to right bottom portion.
