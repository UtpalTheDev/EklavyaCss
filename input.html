Input
Input component is a component that is used to get user input in a text field.
Default Input
Default input text component comes with default width 50% of the parent element and height. To use, just check the below code.
Input text component comes with 3 different sizes-

small- small input component use to take 20% of the parent width. To use this add input sm classes.
medium-medium input component use to take 30% of the parent width. To use this add input md classes.
large-large input component use to take 40% of the parent width. To use this add input lg classes.
Input Error
For showing error in input you just need to add input input-error classes.
Textbox
Default textbox component comes with default width 80% of the parent element and height. To use, just check the below code.
Textbox component comes with 3 different sizes-

small- small textbox component use to take 20% of the parent width. To use this add textbox sm classes.
medium-medium textbox component use to take 40% of the parent width. To use this add textbox md classes.
large-large textbox component use to take 60% of the parent width. To use this add textbox lg classes.
