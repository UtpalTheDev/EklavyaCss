Avatar
It is used to represent user, and displays the profile picture, initials or fallback icon.
Default Avatar
Default avatar give you a default size rounded image that shows user profilepicture. To use default avatar you need to add avatar class in your html and change the image by changing the image url in src.
Avatar comes with 3 different sizes are small, medium and large. To use small size avatar you need to add avatar sm class in your html. To use medium size avatar you need to add avatar md class in your html. To use large size avatar you need to add avatar lg class in your html.
Avatar with Ring
To use ring with avatar you need to add avatar avatar-ring class in your html.
Avatar with Badge
In some products, you might need show a badge on the right corner of the avatar, what we call a badge. To use badge with avatar you need to add avatar avatar-badge class in your html.
Avatar with Ring and Badge
To use badge and ring with avatar you need add avatar avatar-badge avatar-ring classes in your html.
